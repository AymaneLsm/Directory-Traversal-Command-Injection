# DataRoom Enterprise

## Overview

DataRoom Enterprise is a cybersecurity learning project designed to simulate real-world web application security issues within a business environment.

The application intentionally contains vulnerable functionalities that allow students and security enthusiasts to understand how common attacks are performed and how secure development practices can mitigate them.

---

## Project Goals

- Explore common web security vulnerabilities.
- Understand attacker techniques and exploitation methods.
- Practice vulnerability identification and analysis.
- Learn secure coding approaches to reduce security risks.

---

## Implemented Vulnerabilities

### Directory Traversal

A vulnerable file access mechanism demonstrates how improper path validation can allow unauthorized access to sensitive files stored on the server.

### Command Injection

A vulnerable system command feature demonstrates how unsanitized user input may lead to arbitrary command execution.

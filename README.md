# Directory-Traversal-Command-Injection
Educational web security project demonstrating Directory Traversal and Command Injection vulnerabilities in a simulated enterprise environment using Python Flask.
DataRoom Enterprise is an educational web application developed with Python and Flask to demonstrate common web security vulnerabilities. The project intentionally includes Directory Traversal and Command Injection flaws for learning, testing, and cybersecurity training purposes. It also highlights secure coding practices such as input validation, output encoding, and parameterized queries.
